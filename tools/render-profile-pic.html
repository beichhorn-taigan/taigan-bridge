Taigan Bridge — profile picture exporter

Two variants, three preset sizes each. Click a size button to download a PNG ready for Ko-fi, Buy Me a Coffee, GitHub, etc. The circular preview shows how it'll look after a typical profile-pic circular crop.

Mark only — best for small sizes

Use this when the platform displays profile pics small (notification chips, comment avatars, mentions). The bridge mark stays legible down to 32 px.

Custom: px

With 対岸 — best for full-square display

Use this when the platform shows the full square (Slack workspace icon, GitHub OG image, larger profile cards). At 64 px+ the characters become the recognition handle.

Custom: px

Notes

  • Ko-fi accepts PNG / JPG. They auto-crop to a circle on the public page but show a square in the dashboard. Recommended upload: 400 × 400 or 800 × 800 PNG of the mark-only variant.
  • Buy Me a Coffee displays a circle. Same recommendation.
  • GitHub OG image / repo social preview wants 1280 × 640 — neither variant fits, but the 800 × 800 mark works in either corner of a custom OG image built elsewhere.
  • Slack workspace icon shows the full square. Use the 対岸 variant at 512 × 512.
  • For platforms that need a transparent background, use the SVG sources directly (src/assets/profile-pic.svg / profile-pic-jp.svg) — most platforms accept SVG uploads.
